This project presents an IoT-based Smart Decision Making System for real-time soil monitoring
and agricultural decision support. The system uses an ESP32 microcontroller interfaced with soil
and environmental sensors to measure parameters such as soil moisture, temperature, and nutrients.

Sensor data is transmitted via Wi-Fi to cloud platforms including Blynk IoT and Google Sheets
for real-time monitoring and historical data logging. A custom Android application and a web
dashboard provide user-friendly visualization and alerts to support smart irrigation and
precision farming decisions.

## Repository Structure
- ESP32_Arduino_Code: Firmware for sensor data acquisition and Wi-Fi communication
- Android_App: Mobile application for monitoring and alerts
- Web_Dashboard: Browser-based dashboard for data visualization
- Docs: Project diagrams and documentation

## Technologies Used
- ESP32 Microcontroller
- Arduino IDE
- Blynk IoT Platform
- Google Sheets API
- HTML, CSS, JavaScript
- Android Studio
